VoID Description of UWLSWD Linked Open Cultural Heritage Datasets

A description of RDF datasets describing resources associated with digital collections and RDF vocabularies hosted at the University of Washington. The description uses the Vocabulary of Interlinked Datasets (VoID) ontology, available at http://vocab.deri.ie/void.

Currently seven cultural heritage datasets (rdfs:Class-based partitions) are available:

  • Instances of the SourceResource Class
  • Instances of the Aggregation Class
  • Instances of the WebResource Class
  • Instances of the Collection Class
  • Instances of the RightsStatement Class
  • Instances of the Agent Class
  • Instances of the ProvenanceStatement Class

Digital collections from which the dataset's resources were harvested:

  • https://doi.org/10.6069/uwlib.55.A.3.4#AlaskaYukonPacificExpositionCollection
  • https://doi.org/10.6069/uwlib.55.A.3.4#WilliamBrumfieldRussianArchitectureDigitalCollection
