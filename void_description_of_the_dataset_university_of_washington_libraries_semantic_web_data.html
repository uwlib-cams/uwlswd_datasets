University of Washington Libraries' Semantic Web Data

Linked data describing resources associated with digital collections hosted at the University of Washington

This dataset consists of 6 partitions:

  • Instances of the SourceResource Class
  • Instances of the Aggregation Class
  • Instances of the WebResource Class
  • Instances of the Collection Class
  • Instances of the RightsStatement Class
  • Instances of the Agent Class

Digital collections from which the dataset's resources were harvested:

  • https://doi.org/10.6069/uwlib.55.A.3.4#AlaskaYukonPacificExpositionCollection
  • https://doi.org/10.6069/uwlib.55.A.3.4#WilliamBrumfieldRussianArchitectureDigitalCollection

Links to Alternate Serializations for University of Washington Libraries' Semantic Web Data:

  • RDF/XML
  • Turtle

Version Information:

University of Washington Libraries' Semantic Web Data is not versioned as a whole; the partitions only are versioned. See the data below to determine the current version of each partition.
